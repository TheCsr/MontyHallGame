Steps to Execute the Application:


#Running the backend

Step 1: cd backend
Step 2: pip3 install -r requirements.txt
Step 3: python3 app.py

#Running the front-end

Step 1:cd monty-hall-ui
Step 2:npm install
Step 3:npm start


The Frontend can be accesed usually on your localhost port 3000 (but this depends).

URL : <localhost>:port
